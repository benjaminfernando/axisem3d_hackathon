# Using AxiSEM3D

## Overview

To run AxiSEM3D, you must first create a mesh using SalvusMeshLite.  SalvusMeshLite creates an Exodus mesh file (*.e). This file is specified in the input file inparam.model.yaml file for the solver, AxiSEM3D.
